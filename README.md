# Nepali Date Picker

Svelte Component for Nepali Date Picker

## Install

```
npm install @ulleritech/nepali-date-picker
```

## Usage

How to Use

```javascript
<script>
import { NepaliDatePicker } from '@ulleritech/nepali-date-picker';
</script>

<NepaliDatePicker selectedDate="2079/06/30">
```
